Politique de confidentialité

PK Sticky Notes Chrome Extension

Dernière mise à jour : 26 juin 2026

Cette page est la politique de confidentialité officielle de l'extension Chrome PK Sticky Notes. This page is the official privacy policy for the PK Sticky Notes Chrome extension. An English version is available below the French version.

Données collectées

PK Sticky Notes Chrome ne collecte pas de données personnelles pour un serveur externe. L'extension enregistre uniquement les informations nécessaires au fonctionnement des notes dans votre navigateur :

  • Le contenu des notes que vous créez ;
  • La position, la taille, la couleur et l'état épinglé des notes ;
  • L'URL simplifiée de la page associée à chaque note.

Utilisation des données

Ces données sont utilisées uniquement pour afficher, sauvegarder et restaurer vos notes sur les pages web où vous les avez créées.

Stockage des données

Les notes sont stockées localement dans Chrome avec chrome.storage.local. Les préférences de l'extension peuvent être stockées avec chrome.storage.sync afin de conserver les options utilisateur entre sessions Chrome. L'extension n'envoie pas vos notes à un serveur externe.

Partage et transfert

PK Sticky Notes Chrome ne vend pas, ne loue pas, ne partage pas et ne transfère pas les données utilisateur à des tiers. Les données ne sont pas utilisées pour la publicité, le suivi analytique, le profilage, la solvabilité ou des finalités sans rapport avec la fonctionnalité de base de l'extension.

Cookies et suivi

L'extension n'utilise pas de cookies, de pixels de suivi, d'outils analytiques ou de technologies publicitaires.

Accès et suppression

Vous pouvez supprimer une note directement depuis l'interface de l'extension. Vous pouvez aussi supprimer toutes les données de l'extension en supprimant les données stockées par Chrome pour l'extension ou en désinstallant PK Sticky Notes Chrome.

Permissions Chrome

Les permissions déclarées servent uniquement aux fonctions de notes : stockage local des notes, création de notes depuis l'action de l'extension ou le menu contextuel, et association des notes à la page active.

Modifications de cette politique

Cette politique peut être mise à jour si les fonctionnalités de l'extension changent. La date de mise à jour sera modifiée sur cette page.

Contact

Pour toute question concernant cette politique de confidentialité, contactez : contact@mondary.design.

English Version

Data Collected

PK Sticky Notes Chrome does not collect personal data for an external server. The extension only saves the information required for sticky notes to work in your browser:

  • The content of the notes you create;
  • The note position, size, color, and pinned state;
  • The simplified URL of the page associated with each note.

Use of Data

This data is used only to display, save, and restore your notes on the web pages where you created them.

Data Storage

Notes are stored locally in Chrome using chrome.storage.local. Extension preferences may be stored using chrome.storage.sync to preserve user options between Chrome sessions. The extension does not send your notes to an external server.

Sharing and Transfer

PK Sticky Notes Chrome does not sell, rent, share, or transfer user data to third parties. Data is not used for advertising, analytics tracking, profiling, creditworthiness, or purposes unrelated to the extension's core functionality.

Cookies and Tracking

The extension does not use cookies, tracking pixels, analytics tools, or advertising technologies.

Access and Deletion

You can delete a note directly from the extension interface. You can also delete all extension data by clearing Chrome's stored data for the extension or by uninstalling PK Sticky Notes Chrome.

Chrome Permissions

The declared permissions are used only for note features: local note storage, note creation from the extension action or context menu, and associating notes with the active page.

Changes to This Policy

This policy may be updated if the extension features change. The update date will be changed on this page.

Contact

For questions about this privacy policy, contact: contact@mondary.design.
