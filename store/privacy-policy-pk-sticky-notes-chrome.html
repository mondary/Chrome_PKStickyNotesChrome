Politique de confidentialite - PK Sticky Notes Chrome

Derniere mise a jour : 26 juin 2026

This page is the official privacy policy for the PK Sticky Notes Chrome extension. An English version is available below the French version.

Donnees collectees

PK Sticky Notes Chrome ne collecte pas de donnees personnelles pour un serveur externe. L'extension enregistre uniquement les informations necessaires au fonctionnement des notes dans votre navigateur :

  • le contenu des notes que vous creez ;
  • la position, la taille, la couleur et l'etat epingle des notes ;
  • l'URL simplifiee de la page associee a chaque note.

Utilisation des donnees

Ces donnees sont utilisees uniquement pour afficher, sauvegarder et restaurer vos notes sur les pages web ou vous les avez creees.

Stockage des donnees

Les notes sont stockees localement dans Chrome avec chrome.storage.local. Les preferences de l'extension peuvent etre stockees avec chrome.storage.sync afin de conserver les options utilisateur entre sessions Chrome. L'extension n'envoie pas vos notes a un serveur externe.

Partage et transfert

PK Sticky Notes Chrome ne vend pas, ne loue pas, ne partage pas et ne transfere pas les donnees utilisateur a des tiers. Les donnees ne sont pas utilisees pour la publicite, le suivi analytique, le profilage, la solvabilite ou des finalites sans rapport avec la fonctionnalite de base de l'extension.

Cookies et suivi

L'extension n'utilise pas de cookies, de pixels de suivi, d'outils analytiques ou de technologies publicitaires.

Acces et suppression

Vous pouvez supprimer une note directement depuis l'interface de l'extension. Vous pouvez aussi supprimer toutes les donnees de l'extension en supprimant les donnees stockees par Chrome pour l'extension ou en desinstallant PK Sticky Notes Chrome.

Permissions Chrome

Les permissions declarees servent uniquement aux fonctions de notes : stockage local des notes, creation de notes depuis l'action de l'extension ou le menu contextuel, et association des notes a la page active.

Modifications de cette politique

Cette politique peut etre mise a jour si les fonctionnalites de l'extension changent. La date de mise a jour sera modifiee sur cette page.

Contact

Pour toute question concernant cette politique de confidentialite, contactez : contact@mondary.design.

Privacy Policy - English Version

Last updated: June 26, 2026

Data Collected

PK Sticky Notes Chrome does not collect personal data for an external server. The extension only saves the information required for sticky notes to work in your browser:

  • the content of the notes you create;
  • the note position, size, color, and pinned state;
  • the simplified URL of the page associated with each note.

Use of Data

This data is used only to display, save, and restore your notes on the web pages where you created them.

Data Storage

Notes are stored locally in Chrome using chrome.storage.local. Extension preferences may be stored using chrome.storage.sync to preserve user options between Chrome sessions. The extension does not send your notes to an external server.

Sharing and Transfer

PK Sticky Notes Chrome does not sell, rent, share, or transfer user data to third parties. Data is not used for advertising, analytics tracking, profiling, creditworthiness, or purposes unrelated to the extension's core functionality.

Cookies and Tracking

The extension does not use cookies, tracking pixels, analytics tools, or advertising technologies.

Access and Deletion

You can delete a note directly from the extension interface. You can also delete all extension data by clearing Chrome's stored data for the extension or by uninstalling PK Sticky Notes Chrome.

Chrome Permissions

The declared permissions are used only for note features: local note storage, note creation from the extension action or context menu, and associating notes with the active page.

Changes to This Policy

This policy may be updated if the extension features change. The update date will be changed on this page.

Contact

For questions about this privacy policy, contact: contact@mondary.design.
