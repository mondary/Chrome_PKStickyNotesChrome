# PK Sticky Notes - Description Chrome Web Store

## 🇬🇧 English

Add sticky notes anywhere on web pages. Made by PK-Labs.

Create persistent colorful notes directly on any website. Perfect for research, shopping comparisons, meeting notes, and content annotations.

**Features:**
- Right-click to create notes on any page
- Choose from 18 color palettes
- Drag and resize notes freely
- Notes persist between sessions
- Private and local storage (no cloud)
- Works on all websites

Perfect for researchers, shoppers, students, and anyone who needs to remember things while browsing.
