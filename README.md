# PK Sticky Notes Chrome

![Project icon](icon.png)

[🇫🇷 FR](README.md) · [🇬🇧 EN](README_en.md)

✨ Ajoutez des notes collantes directement sur vos pages web.

## ✅ Fonctionnalités
- **Notes collantes** : Créez des notes sur n'importe quel site web.
- **Persistance** : Les notes sont sauvegardées via l'API de stockage de Chrome.
- **Configuration** : Personnalisez l'extension via la page d'options.
- **Menu Contextuel** : Accès rapide aux fonctionnalités via le clic droit.

## 🧠 Utilisation
1. Cliquez sur l'icône de l'extension pour créer une nouvelle note.
2. Utilisez le menu contextuel (clic droit) pour interagir avec les notes.

## ⚙️ Réglages
Accédez à la page d'options (`options.html`) pour modifier les paramètres de l'extension.

## 🧪 Installation
1. Ouvrez Chrome et allez sur `chrome://extensions/`.
2. Activez le **Mode développeur** en haut à droite.
3. Cliquez sur **Charger l'extension non empaquetée**.
4. Sélectionnez le dossier `src` de ce projet.
